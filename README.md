# Kubernetes as a Virtual Machine Management System

Master's thesis by Emil Karlsson, KTH Royal Institute of Technology, 2024.

Compares KubeVirt against traditional VM management approaches. Part of the Software Engineering of Distributed Systems programme at the School of Electrical Engineering and Computer Science.

## Building the report

The report is written in LaTeX using the `kththesis` document class. Build it from the `report/` directory with your preferred LaTeX toolchain (e.g. `latexmk` or `pdflatex` + `biber`).
